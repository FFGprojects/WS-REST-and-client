Ce dossier contient un WebService REST basé sur le FrameWork Spring Boot. Il est sécurisé vis WS Token.

Le client utilise Angular et contient les éléments d'authentification élémentaires.

Un certificat Token permet de sécuriser les communications. Ce certificat est envoyé au client angular et stocké au sein de la session.

Warning : les MdP ne sont cryptés que par le serveur et passent donc en libre.
